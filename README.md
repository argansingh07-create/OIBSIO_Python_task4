# 🌦️ Python Weather App

This is a simple weather application built using **Python**.
The program allows the user to enter a **city name** and then displays the **current weather information** of that city.

This project uses a weather API to fetch real-time data from the internet and show it to the user in the terminal.

## 🚀 Features

* Takes **city name as input**
* Shows **current temperature**
* Displays **weather condition**
* Beginner-friendly Python code
* Uses a real **Weather API**

## 🛠️ Technologies Used

* Python
* Requests module
* OpenWeatherMap API

## ⚙️ How it Works

1. The user enters the name of a city.
2. The program sends a request to the weather API.
3. The API returns weather data in **JSON format**.
4. Python extracts useful information and displays it to the user.
