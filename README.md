# OIBSIO_Python_task4
Python Weather App 🌦️
This is a simple weather app made using Python.
The program takes a city name from the user and shows the current weather information using an API.
Technologies Used
Python
Requests module
OpenWeatherMap API
How it works
User enters the city name
The program sends a request to the weather API
The API returns weather data
The program displays temperature and weather details
What I Learned
Using the requests module in Python
Basics of APIs
Working with JSON data
